\section{Статическая модель}
\subsection{Полярные координаты}
Система наблюдения:
\begin{itemize}
\item $X \sim \NN(M_X, D_X)$ --- декартовы координаты объекта;
\item $Y = \mathop{cart2pol}(X)+\nu$ --- измерения в полярных координатах с помехами $\nu \sim \NN(M_{\nu}, D_{\nu})$;
\item $X_{inv} = \mathop{pol2cart}(Y)$ --- обратное преобразование измерений.
\end{itemize}

Оценки:
\begin{itemize}
\item $\hat{X} = \hat{X}(Y, X_{inv})$ --- УМНФ по наблюдениям и их обратному преобразованию;
\item $\hat{X}_{inv} = \hat{X}_{inv}(X_{inv})$ --- УМНФ по обратному преобразованию наблюдений;
\item $\hat{X}_{lin} = \hat{X}_{lin}(Y)$ --- УМНФ по наблюдениям (линейная оценка);
\item $\hat{X}_{UT} = \hat{X}_{UT}(Y)$ --- оценка, полученная с помощью Unscented transform.
\end{itemize}
